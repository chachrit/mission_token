Mission Token คืออะไร?
──────────────────────
Mission Token คือระบบที่ให้พนักงานสะสม "Token" โดยการทำภารกิจต่างๆ
ที่องค์กรกำหนดขึ้น  เมื่อสะสม Token ได้แล้ว สามารถนำไปแลกรางวัลได้
ในร้านรางวัลภายในระบบ


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. การเข้าสู่ระบบ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1)  เปิดเว็บเบราว์เซอร์ แล้วไปที่ URL ของระบบ (ถามฝ่าย IT หากไม่ทราบ)
2)  กรอก "รหัสพนักงาน" และ "รหัสผ่าน" ให้ตรงกับที่ใช้ใน JOURNAL Portal
3)  กดปุ่ม "เข้าสู่ระบบ"
4)  ระบบจะพาไปยังหน้า Dashboard ของคุณโดยอัตโนมัติ

    หมายเหตุ:
    - รหัสผ่านเป็นชุดเดียวกับ JOURNAL Portal
    - ถ้าลืมรหัสผ่าน กดลิงก์ "ลืมรหัสผ่าน?" บนหน้าเข้าสู่ระบบ
    - ถ้าปิดหน้าต่างทิ้งไว้นาน 2 ชั่วโมง ระบบจะออกจากระบบให้อัตโนมัติ
      เพื่อความปลอดภัย


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. หน้า Dashboard (หน้าแรกหลังเข้าสู่ระบบ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

หน้านี้เป็น "ภาพรวมทุกอย่างของคุณ" ดูได้ในหน้าเดียว ประกอบด้วย:

  กระเป๋า Token
  ──────────────
  • Token คงเหลือ  — จำนวน Token ที่ใช้แลกรางวัลได้ตอนนี้
  • Token ที่ได้รับทั้งหมด — สะสมมาตลอดตั้งแต่เริ่มใช้ระบบ
  • Token ที่ใช้ไปแล้ว — แลกรางวัลไปแล้วเท่าไหร่

  สถิติส่วนตัว
  ─────────────
  • Token เดือนนี้ — Token ที่ได้รับในเดือนปัจจุบัน
  • อันดับของคุณ — ลำดับในตารางผู้นำ (Leaderboard) ของทีม
  • Streak — จำนวนวันติดต่อกันที่คุณทำภารกิจผ่านมาแล้ว

  ภารกิจที่เปิดอยู่ตอนนี้
  ─────────────────────────
  แสดงรายการภารกิจที่สามารถทำได้ พร้อม Token รางวัล
  กดปุ่ม "ดูภารกิจทั้งหมด" เพื่อไปหน้าภารกิจ

  กิจกรรมล่าสุดของคุณ
  ─────────────────────
  แสดง 6 รายการล่าสุดที่คุณส่งงาน พร้อมสถานะ (รออนุมัติ / ผ่าน / ไม่ผ่าน)

  ตารางอันดับ (Leaderboard)
  ──────────────────────────
  อันดับ 1–5 พนักงานที่สะสม Token สูงสุด (นับจาก Token ที่ได้รับทั้งหมด
  ไม่ใช่ Token คงเหลือ)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. หน้าภารกิจ (Challenges)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

หน้านี้แสดงภารกิจทั้งหมดที่เปิดอยู่  แบ่งเป็น 2 ส่วน:
  • "ภารกิจรอคุณอยู่" — ยังทำได้
  • "เสร็จสิ้นแล้ว"  — ทำไปแล้วหรือรออนุมัติ

วิธีใช้งาน:
  1)  เลื่อนดูการ์ดภารกิจ  แต่ละการ์ดแสดงชื่อภารกิจและ Token ที่ได้รับ
  2)  วางเมาส์บนการ์ด (หรือแตะบนมือถือ) การ์ดจะ "พลิก" แสดงรายละเอียด
  3)  ด้านหลังการ์ดจะแสดงเงื่อนไข, วันสิ้นสุด และปุ่มทำภารกิจ

ภารกิจมี 3 ประเภท:

  ┌─────────────────────────────────────────────────────────────┐
  │  ประเภท Quiz (ตอบคำถาม)                                     │
  ├─────────────────────────────────────────────────────────────┤
  │  เงื่อนไขรับ Token:  ต้องตอบถูกทุกข้อ                       │
  │  โอกาสพลาด:          ทำได้ 1 ครั้งเท่านั้น ไม่มีโอกาสแก้ไข   │
  │                                                              │
  │  วิธีทำ:                                                     │
  │  1. กลับการ์ดเพื่อดูรายละเอียด                               │
  │  2. กดปุ่ม "เริ่มทำ Quiz"                                    │
  │  3. ตอบคำถามทุกข้อในหน้าที่เปิดขึ้น                         │
  │  4. กด "ยืนยันคำตอบ"                                         │
  │  5. ถูกทั้งหมด → ได้ Token ทันที                             │
  │     ผิดข้อใดข้อหนึ่ง → ไม่ได้ Token ทำซ้ำไม่ได้             │
  └─────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────┐
  │  ประเภท Photo (ส่งรูปหลักฐาน)                                │
  ├─────────────────────────────────────────────────────────────┤
  │  เงื่อนไขรับ Token:  HR อนุมัติรูปหลักฐานที่ส่งมา           │
  │  โอกาสพลาด:          ถ้า HR ไม่ผ่าน สามารถส่งใหม่ได้        │
  │                                                              │
  │  วิธีทำ:                                                     │
  │  1. กลับการ์ดเพื่อดูรายละเอียดว่าต้องส่งหลักฐานอะไร         │
  │  2. กดปุ่ม "เลือกรูป" และเลือกไฟล์จากอุปกรณ์ของคุณ         │
  │     (JPG, PNG, GIF, WebP  ขนาดไม่เกิน 5MB)                  │
  │  3. กดปุ่ม "ส่งหลักฐาน"                                     │
  │  4. รอ HR ตรวจสอบและอนุมัติ                                  │
  │  5. เมื่ออนุมัติ → ได้ Token                                  │
  │     ถ้าไม่ผ่าน → มีข้อความแจ้ง สามารถกดส่งใหม่ได้           │
  └─────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────┐
  │  ประเภท Strava (ตรวจกิจกรรมออกกำลังกาย)                      │
  ├─────────────────────────────────────────────────────────────┤
  │  เงื่อนไขรับ Token:  ระบบตรวจกิจกรรมใน Strava ของคุณ       │
  │                       ต้องผ่านเงื่อนไขที่กำหนด               │
  │                       เช่น วิ่ง ≥ 5 กม. ภายในช่วงวันที่      │
  │  โอกาสพลาด:          ถ้าไม่พบกิจกรรม สามารถลองใหม่ได้       │
  │                       เมื่อบันทึกกิจกรรมเพิ่มแล้ว            │
  │                                                              │
  │  ขั้นตอนเบื้องต้น (ทำครั้งแรกครั้งเดียว):                    │
  │  1. ไปที่หน้าโปรไฟล์ กด "เชื่อมต่อ Strava"                  │
  │  2. ยืนยันการเชื่อมต่อผ่าน Strava.com                       │
  │                                                              │
  │  วิธีทำภารกิจ Strava:                                        │
  │  1. บันทึกกิจกรรม (เช่น วิ่ง) ให้ครบตามเงื่อนไขก่อน        │
  │  2. กลับการ์ดในหน้าภารกิจ                                    │
  │  3. กดปุ่ม "ตรวจสอบกิจกรรม Strava"                          │
  │  4. รอสักครู่ (ระบบดึงข้อมูลจาก Strava  ใช้เวลา ~15–30 วิ)  │
  │  5. พบกิจกรรมที่ผ่านเงื่อนไข → ได้ Token ทันที              │
  │     ไม่พบ → ทำกิจกรรมเพิ่มแล้วลองใหม่ได้                   │
  │                                                              │
  │  ข้อควรรู้เกี่ยวกับ Strava:                                  │
  │  - กิจกรรมใน Strava ต้องตั้งเป็น "Everyone" หรือ "Followers" │
  │    (ไม่ใช่ Private) ระบบจึงจะมองเห็นได้                     │
  │  - ระบบตรวจกิจกรรมในช่วงวันที่ภารกิจเท่านั้น               │
  └─────────────────────────────────────────────────────────────┘


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. หน้าร้านรางวัล (Rewards)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

หน้านี้ใช้แลก Token เป็นรางวัลต่างๆ

วิธีแลกรางวัล:
  1)  ดูรายการรางวัลทั้งหมด  แต่ละรายการแสดงราคา Token
  2)  ตรวจสอบว่า Token คงเหลือของคุณ (มุมบนขวา) เพียงพอ
  3)  กดปุ่ม "แลกรางวัล" บนรายการที่ต้องการ
  4)  ระบบจะถามยืนยัน กด "ยืนยัน" เพื่อดำเนินการ
  5)  Token จะถูกหักทันที และ HR จะดำเนินการให้รางวัลต่อไป
  6)  รางวัลที่มีสถานะ "กำลังดำเนินการ" หมายถึงรออยู่ที่ HR
  7)  เมื่อ HR "ส่งมอบ" รางวัลแล้ว สถานะจะเปลี่ยนเป็น "สำเร็จ"

รางวัลที่มีรหัสคูปอง:
  - บางรางวัลมีรหัสคูปอง (เช่น Gift Voucher ออนไลน์)
  - รหัสคูปองจะปรากฏให้เห็นในหน้าประวัติเมื่อ HR ยืนยันการส่งมอบแล้ว
  - ยังไม่ส่งมอบ → ยังไม่เห็นรหัส

หมายเหตุ:
  - รางวัลบางอย่างมีจำนวนจำกัด (มีเลข "สต็อก" แสดงอยู่)
  - เมื่อสต็อกหมด ปุ่มแลกจะปิดอัตโนมัติ
  - Token ที่หักไปแล้ว จะได้คืนก็ต่อเมื่อ HR ยกเลิกรายการเท่านั้น


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. หน้าประวัติ (History)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

หน้านี้แสดงประวัติทุกอย่างที่เกิดขึ้นกับบัญชีของคุณ แบ่งเป็น 2 แท็บ:

  แท็บ "Token"
  ─────────────
  รายการรับ-จ่าย Token ทั้งหมด เช่น
  • ได้รับจากทำ Quiz
  • ได้รับจากส่งรูปผ่านการอนุมัติ
  • ได้รับจากทำกิจกรรม Strava
  • Token ที่หักเมื่อแลกรางวัล
  • Token ที่ HR ปรับเพิ่ม/ลด

  แท็บ "รางวัล"
  ──────────────
  รายการแลกรางวัลทั้งหมดพร้อมสถานะ:
  • รอดำเนินการ — รออยู่ที่ HR
  • สำเร็จ       — HR ส่งมอบให้แล้ว  (ถ้ามีรหัสคูปองจะแสดงตรงนี้)
  • ยกเลิก       — ยกเลิกและ Token คืนกลับ


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6. หน้าโปรไฟล์ (Profile)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

แสดงข้อมูลส่วนตัวทั้งหมด ทำได้ 3 อย่าง:

  เปลี่ยนรูปโปรไฟล์
  ──────────────────
  1. กดที่รูปโปรไฟล์ หรือปุ่ม "เปลี่ยนรูป"
  2. เลือกรูปจากอุปกรณ์  (JPG, PNG, GIF, WebP  ขนาดไม่เกิน 2MB)
  3. กด "อัปโหลด"

  เปลี่ยนรหัสผ่าน
  ──────────────────
  1. กรอกรหัสผ่านปัจจุบัน
  2. กรอกรหัสผ่านใหม่ (อย่างน้อย 8 ตัวอักษร)
  3. กรอกรหัสผ่านใหม่อีกครั้งเพื่อยืนยัน
  4. กด "บันทึก"

  เชื่อมต่อ Strava
  ─────────────────
  (สำหรับภารกิจประเภท Strava เท่านั้น)
  1. เลื่อนลงมาหาส่วน "Strava"
  2. กดปุ่ม "เชื่อมต่อ Strava"
  3. ระบบจะพาไปยัง Strava.com ให้กดอนุญาต
  4. เมื่ออนุญาตแล้วระบบจะพากลับมาอัตโนมัติ
  5. ถ้าต้องการยกเลิก กดปุ่ม "ยกเลิกการเชื่อมต่อ"


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
7. หน้า Strava Dashboard
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

(เข้าถึงได้จาก เมนู > Strava  หรือจากหน้าโปรไฟล์)

หน้านี้ดึงข้อมูลจาก Strava ของคุณมาแสดงในแบบสรุป ประกอบด้วย:

  สถิติสรุป (เลือกช่วงเวลา 7 / 30 / 90 วัน)
  • ระยะทางรวม (กิโลเมตร)
  • เวลาเคลื่อนที่รวม
  • จำนวนกิจกรรม
  • ความสูงสะสม (เมตร)
  • ระยะทางเฉลี่ยต่อครั้ง
  • เพซเฉลี่ย (สำหรับกิจกรรมวิ่ง)

  กราฟกิจกรรม
  • ประเภทกิจกรรมยอดนิยม พร้อมระยะทางรวม
  • ตารางวันที่ (สีเข้มขึ้น = วันที่ออกกำลังมาก)

  รายการกิจกรรมทั้งหมด
  • ชื่อ, ประเภท, ระยะทาง, เวลา, ความสูง, วันที่
  • กดชื่อกิจกรรมเพื่อเปิดดูใน Strava.com ได้

  ประวัติภารกิจ Strava ของฉัน
  • รายการภารกิจ Strava ที่เคยส่ง พร้อมผลลัพธ์และ Token ที่ได้

8. คำถามที่พบบ่อย (สำหรับพนักงาน)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ถาม: ทำ Quiz ไปแล้วตอบผิด สามารถลองใหม่ได้ไหม?
ตอบ: ไม่ได้  Quiz ทำได้ 1 ครั้งต่อภารกิจเท่านั้น อ่านคำถามให้ครบก่อนกดยืนยัน

ถาม: ส่งรูปภาพไปแล้ว HR ไม่ผ่าน ทำอะไรได้บ้าง?
ตอบ: สามารถแก้ไขและส่งใหม่ได้  กลับไปที่หน้าภารกิจ กดการ์ดนั้น
      จะมีปุ่ม "ส่งใหม่" ปรากฏขึ้น

ถาม: Token ใช้แล้วหมดไปเลยหรือเปล่า?
ตอบ: ใช่  Token ที่แลกรางวัลจะถูกหักออก
      แต่ Token ที่ได้รับทั้งหมดยังคงนับรวมอยู่ใน Leaderboard

ถาม: Token ที่ได้รับมีอายุไหม?
ตอบ: ระบบปัจจุบันไม่มีวันหมดอายุ (สอบถาม HR เพื่อยืนยัน)

ถาม: กดปุ่มตรวจสอบ Strava แล้วค้างนาน ปกติไหม?
ตอบ: ปกติ  ระบบต้องดึงข้อมูลจาก Strava.com
      อาจใช้เวลา 15–30 วินาที  ห้ามปิดหน้าต่างระหว่างรอ

ถาม: Strava บอกไม่พบกิจกรรม แต่ทำกิจกรรมไปแล้ว?
ตอบ: ตรวจสอบสิ่งเหล่านี้:
      1. กิจกรรมตั้งเป็น Private อยู่ไหม (ต้องเปลี่ยนเป็น Everyone/Followers)
      2. กิจกรรมอยู่ในช่วงวันที่ภารกิจไหม
      3. ประเภทกิจกรรม (เช่น วิ่ง/ปั่น) ตรงกับที่ภารกิจกำหนดไหม
      4. ระยะทาง/เวลาครบตามเงื่อนไขไหม

บทบาทใน Mission Token
──────────────────────
ระบบมี 4 บทบาท:
  • Employee  — พนักงานทั่วไป  เข้าถึงได้เฉพาะหน้าพนักงาน
  • HR        — อนุมัติงาน, จัดการรางวัล, จัดการพนักงาน
  • IT        — สิทธิ์เหมือน HR (สำหรับทีม IT)
  • Admin     — สิทธิ์สูงสุด  ทำได้ทุกอย่าง รวมถึงเปลี่ยนบทบาทผู้อื่น


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. หน้าจัดการภารกิจ (HR > จัดการภารกิจ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

แสดงรายการภารกิจทั้งหมดในระบบ

การสร้างภารกิจใหม่:
  1. กดปุ่ม "+ เพิ่มภารกิจ"
  2. กรอกข้อมูล:
     • ชื่อภารกิจ  (แสดงให้พนักงานเห็น)
     • รายละเอียด  (อธิบายว่าต้องทำอะไร)
     • ประเภท      เลือก Quiz / Photo / Strava
     • Token รางวัล  (จำนวน Token ที่พนักงานจะได้รับเมื่อผ่าน)
     • วันเริ่มต้น / วันสิ้นสุด
  3. กด "บันทึก"

เพิ่มคำถาม Quiz (สำหรับภารกิจประเภท Quiz):
  1. หลังบันทึกภารกิจแล้ว กด "+ เพิ่มคำถาม"
  2. กรอกคำถามและตัวเลือก A–D พร้อมระบุตัวเลือกที่ถูกต้อง
  3. เพิ่มคำถามได้หลายข้อ  พนักงานต้องตอบถูกทุกข้อ

ตั้งค่า Strava (สำหรับภารกิจประเภท Strava):
  ระบุเงื่อนไขที่ต้องการตรวจสอบ:
  • ประเภทกิจกรรม  เช่น Run (วิ่ง), Ride (ปั่น), Walk (เดิน)
  • ระยะทางขั้นต่ำ  (กิโลเมตร)
  • เวลาขั้นต่ำ    (นาที)
  • ความสูงขั้นต่ำ  (เมตร)  — ไม่ระบุก็ได้

เปิด/ปิดภารกิจ:
  กดปุ่ม Toggle บนรายการเพื่อซ่อน/แสดงภารกิจแก่พนักงาน
  ภารกิจที่ปิดอยู่  พนักงานจะไม่เห็น

ลบภารกิจ:
  กดปุ่ม "ลบ"  ระบบจะลบทุกอย่างที่เกี่ยวข้อง
  รวมถึงประวัติการส่งงานและ Token ที่เคยให้ไปแล้ว
  คำเตือน: การลบเป็นการถาวร ไม่สามารถกู้คืนได้


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. หน้าอนุมัติงาน (HR > อนุมัติงาน)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

หน้านี้แสดงรูปภาพ/หลักฐานที่พนักงานส่งมา  รอการตัดสินใจจาก HR

วิธีอนุมัติ:
  1. คลิกดูรูปภาพที่พนักงานส่งมา
  2. พิจารณาว่าถูกต้องตามเงื่อนไขภารกิจหรือไม่
  3. กรอก "หมายเหตุ" (ไม่บังคับ) เพื่อแจ้งพนักงาน
  4. กด "อนุมัติ"  → พนักงานได้ Token ทันที
     หรือ "ปฏิเสธ" → พนักงานได้รับแจ้งและส่งใหม่ได้

ตัวเลข Badge บนเมนู:
  จำนวนสีแดงถัดจาก "อนุมัติงาน" คือจำนวนรายการที่รออยู่


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. หน้าจัดการรางวัล (HR > จัดการรางวัล)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

จัดการรายการรางวัลทั้งหมดในร้านรางวัล

การเพิ่มรางวัลใหม่:
  1. กดปุ่ม "+ เพิ่มรางวัล"
  2. กรอกข้อมูล:
     • ชื่อรางวัล
     • รายละเอียด
     • อีโมจิไอคอน  (แสดงบนการ์ดรางวัล  เช่น 🎁 ☕ 🏖)
     • หมวดหมู่  (Voucher / วันลา / ของสมนาคุณ / สิทธิพิเศษ / ทั่วไป)
     • ราคา Token  (จำนวน Token ที่ต้องใช้แลก)
     • จำนวนสต็อก  (ว่างไว้ = ไม่จำกัด)
     • รหัสคูปอง  (ถ้ามี — พนักงานจะเห็นเมื่อ HR ส่งมอบแล้ว)
     • วันหมดอายุคูปอง  (ถ้ามี)
  3. กด "บันทึก"

จัดการสต็อก:
  แก้ไขจำนวนที่เหลือได้ตลอดเวลาจากหน้ารายการ

เปิด/ปิดรางวัล:
  กดปุ่ม Toggle เพื่อซ่อนรางวัลจากร้านค้าชั่วคราว
  แนะนำให้ใช้วิธีนี้แทนการลบ เพื่อไม่ให้ประวัติพนักงานหาย

ลบรางวัล:
  ลบได้เฉพาะรางวัลที่ยังไม่เคยมีใครแลก
  ถ้ามีประวัติการแลกแล้ว ระบบจะไม่ให้ลบ (ให้ปิดแทน)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. หน้าคำขอแลกรางวัล (HR > จัดการรางวัล > คำขอแลกรางวัล)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

รายการที่พนักงานกด "แลกรางวัล" รออยู่  HR ต้องมายืนยันหลังจัดเตรียมแล้ว

การส่งมอบรางวัล:
  1. ดูรายการที่รออยู่  แต่ละรายการแสดงชื่อพนักงาน, รางวัล, Token ที่ใช้
  2. เมื่อจัดเตรียมรางวัลพร้อมแล้ว กด "ส่งมอบ"
  3. ใส่หมายเหตุ เช่น "ส่งทางอีเมลแล้ว" หรือ "รับที่ HR ได้เลย"
  4. กด "ยืนยัน"  → สถานะเปลี่ยนเป็น "สำเร็จ"
     พนักงานจะเห็นในประวัติ  และถ้ามีรหัสคูปอง จะแสดงให้เห็นด้วย

การยกเลิกรายการ:
  กด "ยกเลิก"  → Token จะถูกคืนให้พนักงานอัตโนมัติ


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. หน้าจัดการพนักงาน (HR > จัดการพนักงาน)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

แสดงรายชื่อพนักงานทั้งหมด พร้อม Token คงเหลือ

สิ่งที่ HR ทำได้:
  • ค้นหาพนักงานด้วยชื่อหรือรหัสพนักงาน
  • เปิด/ปิดบัญชีพนักงาน  (บัญชีที่ปิดจะเข้าสู่ระบบไม่ได้)
  • ปรับ Token  (เพิ่มหรือหักโดยตรง พร้อมระบุเหตุผล)
  • รีเซ็ตรหัสผ่านให้พนักงาน

สิ่งที่เฉพาะ Admin ทำได้เพิ่มเติม:
  • เปลี่ยนบทบาท (Employee / HR / IT / Admin)
  • ลบบัญชีพนักงาน

หมายเหตุด้านความปลอดภัย:
  - ไม่สามารถเปิด/ปิดหรือเปลี่ยน Role บัญชีของตัวเองได้
  - การปรับ Token ทุกครั้งจะถูกบันทึกในประวัติ Token ของพนักงาน


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6. คำถามที่พบบ่อย (สำหรับ HR)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ถาม: พนักงานได้ Token ไปแล้วต้องการถอนคืน ทำได้ไหม?
ตอบ: ได้  ใช้ฟังก์ชัน "ปรับ Token" ในหน้าจัดการพนักงาน
      ใส่จำนวนเป็นลบ เช่น -50 พร้อมระบุเหตุผล

ถาม: ต้องการซ่อนรางวัลชั่วคราวโดยไม่ลบ ทำได้ไหม?
ตอบ: ได้  ใช้ปุ่ม Toggle เพื่อปิด  พนักงานจะไม่เห็นในร้าน
      เปิดกลับเมื่อต้องการโดยกด Toggle อีกครั้ง

ถาม: ถ้าไม่ยืนยันคำขอแลกรางวัล Token จะถูกหักไปก่อนไหม?
ตอบ: ใช่  Token หักทันทีที่พนักงานกดแลก
      ถ้า HR ยกเลิก Token จะถูกคืนให้พนักงานอัตโนมัติ

ถาม: ภารกิจที่หมดวันแล้วยังแสดงให้พนักงานเห็นไหม?
ตอบ: ไม่  ระบบซ่อนภารกิจที่เลยวันสิ้นสุดให้อัตโนมัติ
      แต่ยังมีอยู่ในหน้า HR  สามารถแก้ไขวันหรือลบได้

ถาม: เพิ่มภารกิจ Quiz แล้วไม่มีคำถาม พนักงานจะเห็นอะไร?
ตอบ: ภารกิจจะแสดงในรายการ แต่เมื่อพนักงานกดทำจะขึ้นว่า
      "ภารกิจนี้ยังไม่มีคำถาม"  ต้องเพิ่มคำถามก่อน
